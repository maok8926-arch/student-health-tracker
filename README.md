## Project Description
The Student Health Tracker is a prototype system designed to help university students monitor their daily health activities such as water intake, sleep hours, and exercise. The system provides reminders and dashboard insights to encourage healthier habits.

## Overview of SDLC Work

### Planning
During the planning phase, the team identified stakeholders, wrote functional and non-functional requirements, and created user stories.

### Design
In the design phase, the team created:
- Use Case Diagram
- Class Diagram
- Wireframes
- Prototype screens

### Testing / Evaluation
The prototype was evaluated through user feedback and usability testing. Feedback was collected and improvement suggestions were documented.
