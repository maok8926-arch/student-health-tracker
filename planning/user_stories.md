# User Stories

## Student User Stories

**US1 – Secure Login**

As a student, I want to log in using my email address and password so that I can securely access my personal health records.

**US2 – Record Health Data**

As a student who cares about health management, I want to quickly enter my water intake, sleep hours, and exercise minutes on my phone so that I can continuously track my lifestyle habits.

**US3 – View Dashboard**

As a student, I want to clearly see how much water I have drunk and how long I have slept today on the dashboard so that I know how far I am from achieving my daily health goals.

**US4 – Receive Reminders**

As a busy student, I want to receive reminders if I forget to record my health data so that I can maintain my health tracking routine.

---

## Administrator User Stories

**US5 – Manage User Accounts**

As an administrator, I want to reset passwords or disable user accounts so that I can support users when they experience technical issues or misuse the system.

**US6 – Update Health Tips**

As an administrator, I want to update health tips and recommendation standards so that students receive the latest and most accurate health guidance.

---

## Health Administration User Story

**US7 – View Health Statistics**

As a health administrator, I want to view anonymous statistical reports of student health data so that the school can identify common health problems and organize relevant health education activities.
